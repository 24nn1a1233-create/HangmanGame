# 🎮 Hangman Game (Python)

A simple command-line Hangman game built using Python.

## Features
- Guess one letter at a time
- Tracks incorrect attempts
- Prevents duplicate guesses
- Displays Game Over or You Won

## Technologies Used
- Python 3

## How to Run

```bash
python hangman.py
```

## Sample Output

```
Word: _ _ _
Enter a letter: a
Correct!

Word: a _ _
Enter a letter: p
Correct!

🎉 You Won!
```
